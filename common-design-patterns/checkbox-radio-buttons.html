When to use

The updated design for checkboxes and radio buttons affords better usability for all Canadians. Any use of checkboxes and radio buttons on Canada.ca should follow this design.

Checkboxes and radio buttons should be used in interactive questions, such as in applications to a service, estimators or smart pages and wizards.

It is encouraged to use interactive questions whenever possible on Canada.ca to help Canadians find and complete their task.

Checkboxes can also be used to indicate checklists, such as for things that must be at hand before making a call or eligibility criteria to meet before applying for a service.

What to avoid

Do not use an interactive pattern with radio buttons and checkboxes if a choice is more complex than answering a single question. Use doormat links to provide navigation to a loosely related set of pages.

For example:

  • use a radio button pattern to ask which type of funding a Canadian is looking for
  • use a doormat links pattern instead to present loosely related tasks related to starting a business.

Doormat links pattern for services and information on Canada.ca

When to use and how to implement

Checkboxes

When to use: Use checkboxes when one or more selections can be made in answer to a question.

When not to use: Do not use these larger checkboxes when you are making a checklist or when there is a single item that can be checked. For these situations, use the small checkboxes below.

Small checkboxes

When to use: Use small checkboxes when making a checklist or when there is a single item that can be checked.

When not to use: Do not use these larger checkboxes when you are making a checklist or when there is a single item that is being checked. For these situations, use the small checkboxes below.

Radio buttons

When to use: Use radio buttons when only one selection can be made.

The small checkbox pattern, also included in this spec should be used in preference to the large checkbox pattern in 2 scenarios:

  • When there are stand alone checkboxes to tick off an agreement or detail
  • In checklists, particularly where the checklist items have longer text associated to each checkbox

    Use the larger checkboxes for questions and wizard patterns

    Changes to impaired driving and cannabis-related crime laws

    New penalties for impaired driving and cannabis-related crimes may affect your immigration status in Canada. If you commit one of these crimes, you may not be able to come to or stay in Canada.

    Info alert

    When to use: Use the info alert to draw attention to a clarification, caveat or status of surrounding content.

    Current status of the call for applications: Closed

    Thank you for your interest in Canada Summer Jobs. Applications are now closed.

    Code 
    			<section class="alert alert-info">
			 <h3>Current status of the call for applications: Closed</h3>
			 <p>Thank you for your interest in Canada Summer Jobs. Applications are now closed.</p>
			</section>

    Success alert

    When to use: Use success alert to draw attention to a successful action, such as with form validation.

    Your request for publicly available data from the list of charities has been successful

    The turnaround time to process your request may take up to four (4) weeks. We will contact you if we need more information.

    Code 
    			<section class="alert alert-success">
			 <h3>Your request for publicly available data from the List of charities has been successful</h3>
			 <p>The turnaround time to process your request may take up to four (4) weeks. We will contact you if we need more information.</p>
			</section>
